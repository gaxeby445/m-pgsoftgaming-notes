# m-pgsoftgaming-notes

这是一个用于归档和发布多个独立 HTML 页面的仓库。页面内容不针对特定域名或网站，主要用于整理和展示与相关主题有关的静态页面资源。

## 项目简介

本仓库存储了一系列独立的 HTML 文件，这些文件可能包含说明、参考信息、操作指引或其他静态内容。每个页面均为自包含的 HTML 文档，可直接在浏览器中打开查看。

仓库本身不承担托管或推广任何特定网站的功能，仅作为页面存档和版本管理的工具。

## 目录说明

```
m-pgsoftgaming-notes/
├── pages/           # 存放所有 HTML 页面文件
├── assets/          # 页面引用的静态资源（如图片、样式表、脚本等）
├── docs/            # 额外的文档或说明文件
└── README.md        # 本文件
```

- `pages/`：主要页面目录，每个 HTML 文件代表一个独立的归档页面。
- `assets/`：页面使用的公共资源，按子文件夹分类存放。
- `docs/`：可选，用于存放与页面内容相关的补充文档。

## 页面归档说明

- 每个页面均为独立完整的 HTML 文档，不依赖外部服务或数据库。
- 页面命名应简洁明了，避免特殊字符，推荐使用英文或拼音。
- 页面内容应保持中立、客观，不包含营销性语言或诱导性链接。
- 若页面引用外部资源，建议将资源一并放入 `assets/` 目录，或明确标注来源。

## 维护说明

- 如需添加新页面，请将 HTML 文件放入 `pages/` 目录，并在必要时更新 `assets/` 中的资源。
- 修改现有页面时，请保持内容的一致性和准确性。
- 若页面内容已不再适用，可将其移入 `docs/` 下的 `archive/` 子目录，或在文件名中添加 `_deprecated` 后缀。
- 建议每次提交时附带简要说明，方便追踪变更历史。

## 许可证

本仓库内容仅供个人归档与学习参考使用，不构成任何形式的服务或承诺。请遵守相关法律法规，合理使用页面内容。
